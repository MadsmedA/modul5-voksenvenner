Knapper

Standard knap

Bruges når...

HTML:
<button>Tilmeld</button>
CSS:
.knapper-container button {
background-color: var(--sec-color);
border: none;
margin: 10px 0;
width: fit-content;
padding: 10px 15px;
color: white;
cursor: pointer;
}
