Typografi

Import af typografi

Fonten 'Source Sans Pro' bruges til samtlige typer tekst.

HTML: ---
CSS: @import url('https://fonts.googleapis.com/css2?family=Source+Sans+Pro&display=swap'); body { font-family: 'Source Sans Pro', Arial, sans-serif; }

Header 1

Bruges udelukkende til forsidens titel.

HTML: <h1>Lorem ipsum dolor sit amet.</h1>
CSS: h1 font-size: 50p color: #e30c1 }

Side-overskrift

Bruges som overskrift til hver underside.

HTML: <h2>Lorem ipsum dolor sit amet.<h2>
CSS: h2 font-size: 35p color: #00a0d }

Afsnit-overskrift

Bruges som overskrift til afsnit på en underside.

HTML: <h3&Lorem ipsum dolor sit amet.</h3>
CSS: .knapper-container button.search {background-color: var(--prim-color);}
